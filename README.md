📝 Offline Notes App (SwiftUI + SwiftData)
------------------

A modern offline-first Notes application built using SwiftUI and SwiftData.
This app allows users to efficiently manage notes with full CRUD operations — all stored locally on the device.

🚀 Features
------------------

✅ Create new notes
📖 Fetch & display saved notes
✏️ Edit existing notes
🗑️ Delete notes
💾 Offline storage using SwiftData
⚡ Fast and lightweight performance
🎯 Clean and minimal SwiftUI-based UI

🛠️ Tech Stack
------------------

Language: Swift
UI Framework: SwiftUI
Persistence: SwiftData
Architecture: MVVM (if you used it, otherwise remove this)

📂 Project Structure
------------------

OfflineNotes/
